文件目录基本命令

  • cd：切换目录
  • ./：当前目录
  • cd.. ：返回上一级的目录
  • cd ~：回到当前的用户目录

  • cat ：查看文件,-n显示行号
  • grep：查看文件中符合条件的字符串，一般配合管道符|使用
  • history ：查看历史执行过的指令，后面可跟编号查看最近的几条命令
  • touch+文件名：创建一个文件
  • mkdir+文件名：创建文件夹，要想创建多级目录需要加上-p，例如mkdir -p test1/test2
  • rmdir+文件名：删除文件夹，要想删除多级目录需要加上-p，例如rmdir -p test1/test2
  • echo：输入字符串，也可以输入到文件中！
  • less ：可以分屏查看大文件内容
  • head：用于查看文件开头部分的前10行，如果想查看任意行数，head -n 行数
  • tail：显示文件尾，用法与head相同
  • 两个箭头代表追加，一个箭头代表覆盖。
  • cp ：复制文件或者目录，cp 原来的地方 新的地方
  • -r ：整个文件夹复制
  • \cp ：强制覆盖文件

tar -zxvf +文件名：解压文件

tar -zcvf +文件名 文件名：压缩文件

  • ls：列出目录

-a：all,查看全部的文件，包括隐藏文件

-l ：列出所有的文件，包含文件的属性和权限，没有隐藏文件

两个可以组合使用

  • mv：移动文件或者目录！重命名文件（如果在同一个目录下就是重命名文件，移动后加文件名就是移动并重命名）

-f：强制

-u: 只替换已经更新过的文件

移动目录：例 mv /opt/bbb /home

  • rm：移除文件或目录

-f：忽略不存在的文件，不会出现警告，强制删除

-r：递归的删除目录

-i：互动，删除询问是否删除

rm -rf/*：删除系统中的所有文件

硬链接：A–B，假设B是A的硬链接，那么他们两个指向了同一个文件！，允许一个文件拥有多个路径，用户可以通过这种机制建立硬链接到一些重要文件上，防止误删！

软连接：类似快捷方式，删除了源文件，快捷方式也访问不了了。

删除f1之后查看f2和f3的区别

f2硬链接还在，f3软链接不在了（快捷方式失效）

挂载实际上就是文件系统和硬盘分区之间的一种关系，如果他们关联上了，我们就认为挂载了，如果他们取消了这个关联，就是卸载了。

磁盘分区命令

fdisk /dev/sdb(分区名)

m：显示命令列表

p：显示磁盘分区

n：新增分区

d：删除分区

w：写入并退出

说明：开始分区后输入n，新增分区，然后选择p，分区类型为主分区。两次回车默认剩余全部空间。最后输入w写入分区并退出，若不保存退出输入q。

对分区进行格式化，分配一个UUID。

mkfs -t ext4 /dev/sdb1(分区名)

之后挂载

mount /dev/sdb1 /newdisk(挂载的目录)

umount /dev/sdb1或者umount /newdisk

注意：用命令行挂载重启后会失效，需要使用永久挂载。

永久挂载

通过修改etc/fstab实现挂载，添加完成以后执行mount -a或者reboot重启生效。

Linux组

在linux里每一个用户都必须属于一个组，在linux里每一个文件都有三个概念，谁创建了这个文件，谁就是这个文件的所有者；这个文件是属于哪个组的，那么这个组里的用户就对他拥有一定的权限；其他组对文件也有一定的权限。

进程管理

进程为每个执行的程序，当程序运行到内存中，在内存中运行的程序就是为进程。

ps命令

用来查看当前系统中，有哪些正在执行，以及它们执行的情况。可以不加任何参数。

ps -a：显示当前终端的所有进程信息

ps -u：以用户的格式显示进程信息

ps -x：显示后台进程运行的参数

一般为ps -aux三个结合使用。

ps -aux | more

终止进程kill或killall
