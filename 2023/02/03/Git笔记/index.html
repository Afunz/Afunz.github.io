在IDEA中使用git

红色表示未加入暂存区

此时变更为绿色

或

提交到本地库

写提交信息

此时可以看见提交了本地库，创建了master分支

代码进行了修改时会变成蓝色，此时可以添加到暂存区，也可以直接点提交本地库

版本切换

头指针指向版本，之后进行修改时，创建新的分支，作为修改分支

此时为hot-fix分支

如果hot-fix分支代码修改，切换回master分支会发现代码不同

当hot-fix分支修改完代码，提交后，切换回master分支进行合并分支

产生冲突，合并冲突

合并成功，之后可以自行选择是否删除多余的分支。
